---
id: show-databases
title: SHOW DATABASES
---

Shows the list of databases that exist on the instance.

## Syntax

```
SHOW DATABASES
```

## Examples

```
mysql> SHOW DATABASES;
+----------+
| name     |
+----------+
| default  |
| for_test |
| local    |
| system   |
| test     |
+----------+
```
