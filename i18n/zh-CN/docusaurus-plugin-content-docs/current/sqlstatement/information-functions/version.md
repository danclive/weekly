id: version
title: VERSION
---

Return the current version information of FuseQuery.

## Syntax

```
SELECT VERSION()
```

## Examples

```
mysql> SELECT VERSION();
+-------------------------------------------------------------------------------+
| VERSION()                                                                     |
+-------------------------------------------------------------------------------+
| FuseQuery v-0.1.0-3afb26c(1.54.0-nightly-2021-06-09T07:56:09.461981495+00:00) |
+-------------------------------------------------------------------------------+
```
